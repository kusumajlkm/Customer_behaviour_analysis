#data analytics project showing customer behaviour using postgre sql, powerbi,python


## Overview

This project demonstrates an end-to-end **data analytics workflow**, from raw data ingestion to actionable insights and visualization. It showcases practical skills in Python, SQL, data cleaning, exploratory data analysis (EDA), business intelligence, and reportingâ€”designed to be **recruiter-friendly and easy to understand**.

The goal of the project is to analyze a dataset, extract meaningful insights, and present them through an interactive **Power BI dashboard** and a concise **PPT report**.

---

## Dataset

* **Source**: CSV / Excel file (can be replaced with any structured dataset)
* **Type**: Structured tabular data
* **Content**: Contains business-related attributes such as sales, customers, products, dates, and revenue
* **Size**: Medium-sized dataset suitable for EDA and SQL analysis

---

## Tools & Technologies

* **Python**: Data loading, cleaning, and EDA

  * Libraries: `pandas`, `numpy`, `matplotlib`, `seaborn`
* **SQL**: Data querying and analysis

  * Databases: PostgreSQL / MySQL / SQL Server
* **Power BI**: Interactive dashboard creation
* **MS PowerPoint**: Insight presentation and reporting
* **Jupyter Notebook**: Development and analysis environment

---

## Steps Performed

1. Loaded the dataset using Python
2. Performed Exploratory Data Analysis (EDA)

   * Summary statistics
   * Missing value analysis
   * Trend and distribution analysis
3. Cleaned and transformed data

   * Handled missing and inconsistent values
   * Data type corrections
4. Stored and queried data using SQL

   * Filtering, aggregation, joins, and subqueries
5. Built an interactive Power BI dashboard
6. Created a PPT report summarizing key insights and business impact

---

## How to Run the Project

1. Clone the repository

   ```bash
   git clone <repository-url>
   ```
2. Install required Python libraries

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook and run cells step-by-step
4. Import cleaned data into PostgreSQL/MySQL/SQL Server
5. Execute the provided SQL queries
6. Open the Power BI file to explore the dashboard
7. Review the PPT report for summarized insights
